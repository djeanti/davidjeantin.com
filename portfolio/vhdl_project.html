The objective of this project is to design on FPGA different architectures implementing three different algorithms for the calculation of the square root with different design constraints. These architectures are designed and tested using VHDL. These circuits will calculate the square root of an unsigned input A with a size of 2n bits.

A total of three algorithms are used:

- The first is based on Newton's method

- The second and third are iterative algorithms that successively calculate each of the bits of the result starting with the MSB.
