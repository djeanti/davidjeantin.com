Implementation of a template matching program in C++ using OpenCV to rebuilt a fresco. The approach used for template matching was area-based.

The project was divided in three parts:

- Creation of the "likelihood map" for a fragment in order to estimate the general location of the fragment inside the original fresco. A "backprojection" of the fresco is created specifically for each fragment. The white areas have a higher chance of corresponding to the fragment location while its the opposite for the darker areas.

- Usage of keypoints in order to get the exact location based on the backprojection previously calculated.

- Once we know the fine position for each fragment we can put them back inside a new empty fresco. The only parameter left is to know the correct rotation of each fragment, for that we use a projection matrix.

All the functionnalities mentionned above where implemented thanks to OpenCV library (For the backprojection, and the BRISK algorithm for instance).
