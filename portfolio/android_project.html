Creation of a video game in Android Studio using Java. Creation of UML diagrams during this process. This game was coded in Java and features a hero who must save the princess by evading the bullets until the last stage. A message saying "You Win" appears on screen if the hero succeeds, in the latter, we go back to the main menu after showing a message saying "Game Over" on screen. GitHub link.

Download the game to play it yourself!
