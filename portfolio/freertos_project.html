Development of a remote communicating temperature indicator using FreeRTOS. The indicator, built around an ESP32, has an OLED display, an asynchronous serial link and an interface with a CAN bus. This CAN bus has been used to link with a temperature sensor module. The temperature sensor module consists of an LM75B sensor located on an LPC1768 mbed expansion board.
