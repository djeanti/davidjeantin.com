Calculator in Python
This project consisted of creating a simple GUI calculator in Python using the tkinter package. The calculator have the basics four operators (+,-,*,/). The application is able to reset the value on its screen (C button) and also have other operators : one for to change the sign (+/-) and also the modulo operator (%).
