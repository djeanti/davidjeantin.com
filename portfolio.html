UNIVERSITY OF PARIS-SACLAY
Cobotic project

FANUC, the largest maker of industrial robots in the world, organizes each year a competition for robotics enthusiasts. I participated in this competition thanks to my school : Polytech Paris-Saclay. This year the subject lends itself around an assembly line motors (each weighing 5.1 Kg). We had to conceive an entire work environment where the operator musn't have musculoskeletal disorders. My team was composed of two other people aside from me and my role, as an IT manager, was to create a simulation of our solution using a software called RoboGuide.

RFID project

UNIVERSITY OF PARIS-SACLAY
FreeRTOS project

Development of a remote communicating temperature indicator. The indicator, built around an ESP32, has an OLED display, an asynchronous serial link and an interface with a CAN bus. This CAN bus has been used to link with a temperature sensor module. The temperature sensor module consists of an LM75B sensor located on an LPC1768 mbed expansion board.

UNIVERSITY OF PARIS-SACLAY
ZUMO project

This project, done at Polytech Paris-Saclay, consist of using a robot: the Zumo 32U4 to fight in an arena with other robots (of the same type) while respecting certain criteria. We were in a team of 4 people. We had, in fact, to program the robots pour so that they can follow a line leading ot the arena. By using event-driven programming we produced a specific behaviour of the robot in some situations and we tried to created the best strategy to win against the other robot in the arena since this was to final goal. This project nurtured my skills as an engineer in electronics and informatics like, for instance, to read the datasheet of a composant.
